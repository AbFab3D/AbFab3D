The Format Menu

The Format menu contains menu items to change the look and feel of the current RText session. It contains the following menu items:


Action What It Does
Word Wrap Toggles word wrap.
Background... Lets you choose a color or picture for the background of all text areas.
Look & Feel Displays a submenu allowing you to select the active Swing Look and Feel.
Options... Displays the Options dialog, where virtually everything in RText can be configured.
