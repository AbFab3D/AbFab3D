Introduction

RText is a customizable programmer's text editor written in Java. Some of its features include:

  • Syntax highlighting
  • Macro support
  • CTags support
  • Editing multiple documents at once
  • Printing and print preview
  • Find/replace/find in files dialogs
  • Undo/redo
  • Completely customizable user interface
  • Online help

Most people should be able to pick up RText and get started using it right away, but if you need any help, well, that's why this Help is here. Please visit the FAQ and Bug List first, as they'll most likely answer any questions you may have.
