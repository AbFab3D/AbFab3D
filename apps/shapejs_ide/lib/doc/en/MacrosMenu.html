The Macros Menu

The Macros menu contains menu items to record and play back macros. It contains the following menu items:


Action What It Does
Begin Recording Macro Starts the recording of a macro.
Record Temporary Macro Starts the recording of a temporary macro.
Stop Recording Macro Stops recording a macro, if one is being recorded.
Playback Last Macro Plays back the last macro.
Load Macro Displays a submenu allowing you to select a submenu to load for playback.
