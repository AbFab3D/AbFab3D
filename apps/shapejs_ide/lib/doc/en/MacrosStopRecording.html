Stop Recording a Macro

To stop recording a macro, temporary or not, do one of the following:

  1. Go to the Macros menu, then select Stop Recording Macro.
  2. Press Ctrl+Shift+S.

The cursor will change back to the default when hovered over text documents. Also, if you were NOT recording a temporary macro, a dialog will appear prompting you for a name for the macro. After you name it, this macro will appear in the Load Macro submenu of the Macros menu, and can be removed via the Options dialog.

Temporary or not, the just-recorded macro can now be played back, as it is the "last macro."

See also: Recording macros, Recording temporary macros, Playing back macros
