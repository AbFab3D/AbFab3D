The Menu Bar

One of the most easily-recognizable GUI components, the menu bar is the collection of pulldown menus at the top of the RText window. RText currently has the following menus available:

  • File Menu
  • Edit Menu
  • Format Menu
  • View Menu
  • Window Menu (only available when using the MDI view style)
  • Macros Menu
  • Help Menu
