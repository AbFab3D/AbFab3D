Saving All Open Text Documents

You can save all currently-active text documents any of the following ways:

You can save the currently-active text document in one of three ways:

  1. Click on the 'Save All' button on the Toolbar. If a document has not been previously saved, you will be prompted for a file name.
  2. Go to the File menu, then select 'Save All'. If a document has not been previously saved, you will be prompted for a file name.
  3. Define a shortcut for this action and use it.

See also: Saving the Current Text Document, Saving the Current Text Document with a New File Name
