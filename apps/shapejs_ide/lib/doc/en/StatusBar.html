Status Bar

The Status Bar is displayed at the bottom of the main window. You can toggle the display of the Status Bar by selecting "Status Bar" from the View menu.

The Status Bar has the following components:

  1. A status line. As you move your mouse over Toolbar buttons and menu items, the status line will give a description of what the items do.
  2. A Read Only mode indicator. This tells you if the currently active text document is Read Only.
  3. A Caps Lock indicator. This tells you whether or not Caps Lock is enabled.
  4. An Insert/Overwrite mode indicator. If you are in Insert mode, "OVR" will be grayed out; if you are in Overwrite mode, "OVR" will be activated.
  5. A "Macro is Recording" indicator (REC). If you are in the process of recording a Macro, this indicator will be activated.
  6. A line/column indicator. If word wrap is disabled, you will see the line number and column number of the current cursor position.
